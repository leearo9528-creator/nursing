# Nurse Avengers — 간호사 진로 특강 웹앱

중학생 대상 2차시(90분) 간호사 진로 특강 인터랙티브 웹앱입니다.

## 파일 구조

```
index.html    학생용 (모바일 390px 기준)
app.js        학생용 게임 데이터·로직
teacher.html  선생님용 (1920×1080 모니터/프로젝터)
teacher.js    선생님용 슬라이드·진행 로직
vercel.json   배포 설정
```

## Vercel 배포

```bash
npm i -g vercel   # 최초 1회
vercel            # 배포 (프로젝트 루트에서 실행)
```

배포 후 생성되는 URL을 `teacher.html`의 QR 코드가 자동으로 표시합니다.

## 수업 진행 방법

### 선생님 화면 (`/teacher.html`)
- 교실 앞 모니터/프로젝터에 띄웁니다
- `→` / `Space` 로 슬라이드를 넘깁니다
- 우측 사이드바 QR 코드를 학생이 스캔해 접속합니다
- 좌측 사이드바에서 수업 흐름·타임라인을 확인합니다

### 학생 화면 (`/index.html`)
- 학생이 개인 스마트폰으로 QR 스캔 후 이름 입력
- 각자 자기 속도로 진행 (실시간 동기화 없음)
- 진행 상태·점수가 `localStorage`에 자동 저장됨

## 점수 체계

- OX 퀴즈 정답: **+10pt**
- 3연속 정답 보너스: **+5pt**
- 상황극 공감 점수: 선택에 따라 **0~+10pt**
- 트리아지 정답: **+15pt**

| 랭크 | 기준 |
|------|------|
| 🦸 NURSE-AVENGER | 300pt 이상 |
| 💉 SPECIALIST | 220pt 이상 |
| 🩺 TRAINEE | 140pt 이상 |
| 🩹 ROOKIE | 0pt 이상 |

## 로컬 실행

별도 빌드 없이 아무 정적 서버로 실행 가능합니다.

```bash
# Python
python -m http.server 3000

# Node.js (npx)
npx serve .
```

브라우저에서 `http://localhost:3000` 접속.
